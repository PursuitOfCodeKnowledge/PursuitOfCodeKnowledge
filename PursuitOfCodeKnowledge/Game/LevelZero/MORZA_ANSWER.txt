Чтобы расшифровать файл для начала нужно установить Python:
sudo apt update
sudo apt install python3
После установки запустить интерпретатор в терминале:
python3
Затем:
exec(open('MorzaDecrypt.py').read())
Затем ввести:
exit()
или нажать клавиши на клавиатуре ctrl+D и ты выйдешь из интерпретатора.
После этого набери команду:
ls
В ней ты увидишь новый появившийся файл LawOfGod_decrypt.txt
Набираешь на клавиатуре новую команду:
cat LawOfGod_decrypt.txt
или
nano LawOfGod_decrypt.txt
Затем сумируем все цифры и получаем 55
Это и будет пароль к архиву Universe1, распаковать архив можно командой:
7z x Universe1.7z -p55
После распаковки архива появится директория Universe1, переходим к этой директории:
cd Universe1
И дальше смотрим какие здесь есть директории в Universe1 при помощи команды:
ls
MilkyWayGalaxy
cd MilkyWayGalaxy
ls
PlanetEarth
cd PlanetEarth 
ls
SifirisLumoscifr
cd SifirisLumoscifr
или
cd MilkyWayGalaxy/PlanetEarth/SifirisLumoscifr
