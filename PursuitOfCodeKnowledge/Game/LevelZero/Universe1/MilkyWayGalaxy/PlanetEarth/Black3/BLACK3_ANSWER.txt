Чтобы выполнить задание:
Первое нужно посмотреть где ты находишься, используй команду pwd:
pwd
К примеру она тебе покажет путь /home/sergey/Desktop/PursuitOfCodeKnowledge.git/PursuitOfCodeKnowledge/Game/LevelZero/Universe1/MilkyWayGalaxy/PlanetEarth/Black3, получается ты сейчас находишься в Black3, 
тебе нужно вернуться назад используй команду cd ..:
cd ..
После этого набери команду ls или ls -a и посмотри что находится вокруг тебя какие файлы и директории:
ls
или
ls -a
Затем набери команду mkdir чтобы создать директорию Black (в том случае если ещё не создавал директорию Black):
mkdir Black
Затем перейди директорию в Black3:
cd Black3
Потом перенеси файл BLACK3.txt в директорию Black:
mv BLACK3.txt ../Black/
После скопируй файл файл BLACK3_ANSWER.txt в директорию Black:
cp BLACK3_ANSWER.txt ../Black/
Затем проверь что скопировал файл BLACK3_ANSWER.txt в директорию Black при помощи команды ls:
ls ../Black/BLACK3_ANSWER.txt
Потом удали файл BLACK3_ANSWER.txt при помощи команды rm:
rm BLACK3_ANSWER.txt
или
rm -i BLACK3_ANSWER.txt
или
rm -f BLACK3_ANSWER.txt
После выйди из этой директории Black3:
cd ..
Затем удали директорию Black3:
rm -r Black3
После убедись что удалил директорию:
ls 
или
ls -a
Если исследовал уже директорию Black2, то установи пакет lynx:
sudo apt-get update
sudo apt-get install lynx
При установке пакета lynx если спросит что-то, нажми клавишу y и нажми enter

Основная навигация:
Когда вы откроете Lynx, вы увидите текстовую версию веб-страницы. Основные клавиши для навигации:
    Стрелки вверх и вниз: Перемещение между ссылками на странице.
    Стрелка вправо (или Enter): Открыть выделенную ссылку.
    Стрелка влево: Вернуться на предыдущую страницу.
    'G': Открыть диалог для ввода нового URL. Это позволяет вам переходить к любому веб-сайту, который вы хотите.
    'Q': Выйти из Lynx.
    'H': Открыть справку с информацией о клавиатурных сочетаниях и настройках.
     /: Вы можете найти текст на странице, нажав /, а затем введя искомое слово или фразу. Этот поиск осуществляется по тексту, представленному на текущей странице.

Basic navigation:
When you open Lynx, you will see a text version of the webpage. The main keys for navigation:
    Up and down arrows: Move between links on the page.
    Right arrow (or Enter): Open the selected link.
    Left arrow: Return to the previous page.
    'G': Open a dialog to enter a new URL. This allows you to navigate to any website you want.
    'Q': Log out of Lynx.
    'H': Open the help with information about keyboard shortcuts and settings.
     /: You can find the text on the page by clicking / and then entering the word or phrase you are looking for. This search is based on the text provided on the current page.
