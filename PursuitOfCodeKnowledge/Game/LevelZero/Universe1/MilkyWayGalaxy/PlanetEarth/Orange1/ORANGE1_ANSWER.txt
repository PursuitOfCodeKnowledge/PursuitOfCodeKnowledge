Чтобы выполнить задание:
Первое нужно посмотреть где ты находишься, используй команду pwd:
pwd
К примеру она тебе покажет путь /home/sergey/Desktop/PursuitOfCodeKnowledge.git/PursuitOfCodeKnowledge/Game/LevelZero/Universe1/MilkyWayGalaxy/PlanetEarth/Orange1, получается ты сейчас находишься в Orange1, 
тебе нужно вернуться назад используй команду cd ..:
cd ..
После этого набери команду ls или ls -a и посмотри что находится вокруг тебя какие файлы и директории:
ls
или
ls -a
Затем набери команду mkdir чтобы создать директорию Orange (в том случае если ещё не создавал директорию Orange):
mkdir Orange
Затем перейди директорию в Orange1:
cd Orange1
Потом перенеси файл ORANGE1.txt в директорию Orange:
mv ORANGE1.txt ../Orange/
После скопируй файл файл ORANGE1_ANSWER.txt в директорию Orange:
cp ORANGE1_ANSWER.txt ../Orange/
Затем проверь что скопировал файл ORANGE1_ANSWER.txt в директорию Orange при помощи команды ls:
ls ../Orange/ORANGE1_ANSWER.txt
Потом удали файл ORANGE1_ANSWER.txt при помощи команды rm:
rm ORANGE1_ANSWER.txt
или
rm -i ORANGE1_ANSWER.txt
или
rm -f ORANGE1_ANSWER.txt
После выйди из этой директории Orange1:
cd ..
Затем удали директорию Orange1:
rm -r Orange1
После убедись что удалил директорию:
ls 
или
ls -a
