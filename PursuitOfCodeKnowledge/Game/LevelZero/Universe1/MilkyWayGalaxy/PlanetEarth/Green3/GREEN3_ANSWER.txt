Чтобы выполнить задание:
Первое нужно посмотреть где ты находишься, используй команду pwd:
pwd
К примеру она тебе покажет путь /home/sergey/Desktop/PursuitOfCodeKnowledge.git/PursuitOfCodeKnowledge/Game/LevelZero/Universe1/MilkyWayGalaxy/PlanetEarth/Green3, получается ты сейчас находишься в Green3, 
тебе нужно вернуться назад используй команду cd ..:
cd ..
После этого набери команду ls или ls -a и посмотри что находится вокруг тебя какие файлы и директории:
ls
или
ls -a
Затем набери команду mkdir чтобы создать директорию Green (в том случае если ещё не создавал директорию Green):
mkdir Green
Затем перейди директорию в Green3:
cd Green3
Потом перенеси файл GREEN3.txt в директорию Green:
mv GREEN3.txt ../Green/
После скопируй файл файл GREEN3_ANSWER.txt в директорию Green:
cp GREEN3_ANSWER.txt ../Green/
Затем проверь что скопировал файл GREEN3_ANSWER.txt в директорию Green при помощи команды ls:
ls ../Green/GREEN3_ANSWER.txt
Потом удали файл GREEN3_ANSWER.txt при помощи команды rm:
rm GREEN3_ANSWER.txt
или
rm -i GREEN3_ANSWER.txt
или
rm -f GREEN3_ANSWER.txt
После выйди из этой директории Green3:
cd ..
Затем удали директорию Green3:
rm -r Green3
После убедись что удалил директорию:
ls 
или
ls -a
Если исследовал уже директорию Green2, то заходим в своё окружение(смотри директорию Blue1):

cd LearnPython

cd ListAndTheirMethods
