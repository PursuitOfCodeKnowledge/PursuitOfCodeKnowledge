Чтобы выполнить задание:
Первое нужно посмотреть где ты находишься, используй команду pwd:
pwd
К примеру она тебе покажет путь /home/sergey/Desktop/PursuitOfCodeKnowledge.git/PursuitOfCodeKnowledge/Game/LevelZero/Universe1/MilkyWayGalaxy/PlanetEarth/Green5, получается ты сейчас находишься в Green5, 
тебе нужно вернуться назад используй команду cd ..:
cd ..
После этого набери команду ls или ls -a и посмотри что находится вокруг тебя какие файлы и директории:
ls
или
ls -a
Затем набери команду mkdir чтобы создать директорию Green (в том случае если ещё не создавал директорию Green):
mkdir Green
Затем перейди директорию в Green5:
cd Green5
Потом перенеси файл GREEN5.txt в директорию Green:
mv GREEN5.txt ../Green/
После скопируй файл файл GREEN5_ANSWER.txt в директорию Green:
cp GREEN5_ANSWER.txt ../Green/
Затем проверь что скопировал файл GREEN5_ANSWER.txt в директорию Green при помощи команды ls:
ls ../Green/GREEN5_ANSWER.txt
Потом удали файл GREEN5_ANSWER.txt при помощи команды rm:
rm GREEN5_ANSWER.txt
или
rm -i GREEN5_ANSWER.txt
или
rm -f GREEN5_ANSWER.txt
После выйди из этой директории Green5:
cd ..
Затем удали директорию Green5:
rm -r Green5
После убедись что удалил директорию:
ls 
или
ls -a
Если исследовал уже директорию Green4, то заходим в своё окружение(смотри директорию Blue1):
