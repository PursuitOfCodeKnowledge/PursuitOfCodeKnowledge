pwd

тебе нужно вернуться назад используй команду cd ..:
cd ..
После этого набери команду ls или ls -a и посмотри что находится вокруг тебя какие файлы и директории:
ls
или
ls -a
Затем набери команду mkdir чтобы создать директорию Blue (в том случае если ещё не создавал директорию Blue):

cd Blue1

cp BLUE1_ANSWER.txt ../Blue/

ls ../Blue/BLUE1_ANSWER.txt
Потом удали файл BLUE1_ANSWER.txt при помощи команды rm:
rm BLUE1_ANSWER.txt
или
rm -i BLUE1_ANSWER.txt

После выйди из этой директории Blue1:

Если исследовал уже директорию Black6,тогда скачай Anaconda из терминала при помощи команды wget:
wget https://repo.anaconda.com/archive/Anaconda3-2024.10-1-Linux-x86_64.sh
Когда скачается файл, вводим команду ls:
ls
И мы должны увидить файл Anaconda3-2024.10-1-Linux-x86_64.sh, после этого назначаем права на исполнение этому файлу при помощи команды chmod:
chmod +x ./Anaconda3-2024.10-1-Linux-x86_64.sh
После того как назначили права запускаем этот файл в терминале:
./Anaconda3-2024.10-1-Linux-x86_64.sh
Во время установки нажимаем enter просматриваем лицензию используюя стрелочку вниз как дойдем до конца документа нажимаем клавишу q, затем напишем yes, потом enter.
Когда установилась Anaconda, активируем её в терминале при помощи source:
source ~/anaconda3/bin/activate
После того как активировали набираем команду conda в терминале:
conda --version
Как увидели версию conda, посмотрим какие есть окружения в conda:
conda env list
или
conda info --envs
Затем можно посмотреть в каком  окружении находишься:
echo $CONDA_DEFAULT_ENV
После как увидели окружение, делаем отключение(деактивацию окружения conda) в терминале: 
conda deactivate
Как вышли из окружения создаем своё окружение envlearnpython в терминале:
conda create --name envlearnpython python=3.12 -y
После создания своего окружения делаем включение(активация) своего окружения:
conda activate envlearnpython
Как включили окружение, что есть в этом окружении:
conda list
После просмотра пакетов в своём окружении, устанавливаем пакеты numpy, networkx, matplotlib:
conda install numpy matplotlib networkx -y
Потом проверяем что установили пакеты:
conda list
После этого проверим в интерпретаторе Python:
python3
Как зашли в интерпретатор пишем:
import numpy
print(numpy.__version__);
Должна отобразиться версия пакета numpy
Затем проверяем версию пакета networkx:
import networkx as nx
print(nx.__version__);
import matplotlib as plt
print(plt.__version__);
Теперь выходим из интерпретатора:
exit()
Затем набираем команду ls:  
ls
И вы должны увидить файл Anaconda3-2024.10-1-Linux-x86_64.sh
Теперь удаляе этот файл при помощи команды rm:
rm Anaconda3-2024.10-1-Linux-x86_64.sh
Убеждаемся что удалили при помощи команды ls:
ls
Затем входим в директорию LearnPython(выполняли задание из Black5):
cd LearnPython
После создаем файл при помощи команды touch:
touch DATATYPES.py
После создания файла открываем его в текстовом редакторе nano в терминале:
nano DATATYPES.py
После открытия файла в редакторе пишем:
