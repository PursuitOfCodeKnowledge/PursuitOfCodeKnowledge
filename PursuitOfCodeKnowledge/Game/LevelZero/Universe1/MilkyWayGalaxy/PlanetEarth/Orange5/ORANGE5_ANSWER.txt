Чтобы выполнить задание:
Первое нужно посмотреть где ты находишься, используй команду pwd:
pwd
К примеру она тебе покажет путь /home/sergey/Desktop/PursuitOfCodeKnowledge.git/PursuitOfCodeKnowledge/Game/LevelZero/Universe1/MilkyWayGalaxy/PlanetEarth/Orange5, получается ты сейчас находишься в Orange5, 
тебе нужно вернуться назад используй команду cd ..:
cd ..
После этого набери команду ls или ls -a и посмотри что находится вокруг тебя какие файлы и директории:
ls
или
ls -a
Затем набери команду mkdir чтобы создать директорию Orange (в том случае если ещё не создавал директорию Orange):
mkdir Orange
Затем перейди директорию в Orange5:
cd Orange5
Потом перенеси файл ORANGE5.txt в директорию Orange:
mv ORANGE5.txt ../Orange/
После скопируй файл файл ORANGE5_ANSWER.txt в директорию Orange:
cp ORANGE5_ANSWER.txt ../Orange/
Затем проверь что скопировал файл ORANGE5_ANSWER.txt в директорию Orange при помощи команды ls:
ls ../Orange/ORANGE5_ANSWER.txt
Потом удали файл ORANGE5_ANSWER.txt при помощи команды rm:
rm ORANGE5_ANSWER.txt
или
rm -i ORANGE5_ANSWER.txt
или
rm -f ORANGE5_ANSWER.txt
После выйди из этой директории Orange5:
cd ..
Затем удали директорию Orange5:
rm -r Orange5
После убедись что удалил директорию:
ls 
или
ls -a
Если исследовал уже директорию Orange4, то заходим в своё окружение(смотри директорию Blue1):
conda activate envlearnpython
Затем заходим в свою директорию LearnPython:
cd LearnPython
Потом создай директорию AlgorithmsAndDataStructures:
mkdir AlgorithmsAndDataStructures
Затем перейди в эту директорию AlgorithmsAndDataStructures:
cd AlgorithmsAndDataStructures
Итак создай файл linear_search.py:
touch linear_search.py
Потом открой файл linear_search.py:
nano linear_search.py
Пишем в файле:
def linear_search(arr, target):
    for i in range(len(arr)):
        if arr[i] == target:
            return i
    return -1

def main():
    robots = ["T1000", "T2000", "T3000", "T4000", "T5000", "T6000", "T7000"]
    target = "T7000"
    print(linear_search(robots, target))
if __name__ == "__main__":
    main()
Нажимаем ctrl + o, затем enter и выходим из файла ctrl + x, после запускаем скрипт:
python3 linear_search.py

После выходим из директории AlgorithmsAndDataStructures:
cd ..
Дальше выйди из папки LearnPython:
cd ..
Потом иследуй другую директорию, например Orange6
cd Orange6
